### How to download from Higgsfield:
1. Go to higgsfield.ai → your generations
2. Right-click each image/video → Save As
3. Rename to match the table above
4. Upload to the `assets/img/` and `assets/vid/` folders in this repo
5. Vercel will auto-redeploy
